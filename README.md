# PAINT
Una aplicación de dibujo inspirada en el Paint. Este proyecto permite a los usuarios dibujar, borrar, personalizar colores y exportar en formato png y jpg.

---

## Características Principales

La aplicación cuenta con las siguientes funcionalidades mínimas requeridas:

*   **Lienzo Interactivo (Canvas
*   **Herramientas de Dibujo:**
    *   **Lápiz:** Para dibujar libremente.
    *   **Borrador:** Para borrar zonas del lienzo.
*   **Selector de Color (Color Picker)
*   **Guardar Dibujo:** Permite exportar y descargar el lienzo.
